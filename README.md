## 使用方法

![](./images/01.png)

1. 先在菜单中设置[百度语音合成API](http://ai.baidu.com/tech/speech/tts)的token（默认的token大概5月份就过期了，所以务必每30天更新一次，否则不能使用）

2. 然后打开“开始夸”窗口，输入夸奖语句。

3. 之后只需保持窗口不关闭，在你操作Sketch时，就会不停播放夸奖的语音鼓励你。
